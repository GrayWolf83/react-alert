# Компонент React Alert

Выполнен в целях практики работы с библиотекой React

### Как это работает?
При нажатии на кнопки:
1. Кнопки становятся не активными за счет изменения state компонента Button;
2. Компонент Alert получает необходимые параметры за счет изменения state;
3. Таймаут 3 секунды;
4. Обнуляется state компонента Alert;
5. Кнопки становятся активными за счет изменения state компонента Button.
