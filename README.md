# Описание приложения

Данное приложение выполнено в целях получения практического опыта в разработке на React


## Как это работает?
1. Если не ввести данные в input, то будет выведен alert с классом 'danger';
2. Если ввести данные в input, то будет выведен alert с классом 'success';
